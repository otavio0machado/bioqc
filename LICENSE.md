## Disclosure regarding this showcase

This repository is a **sanitized portfolio version** of a system originally
built for a clinical laboratory client. The source has been published with the
client's explicit written authorization. The following changes were made
relative to the production codebase:

- All institutional branding, names, contact details, and identifying
  information replaced with the fictitious brand "BioQC".
- Java package renamed from the client-specific namespace to `com.bioqc`.
- Internal planning documents, audit reports, and project management
  artifacts removed.
- All seed and demo data is fictitious. No real patient, staff, or
  laboratory data is present.

The author retains the original engagement under a separate license with the
client; the open-source MIT license above applies only to this sanitized
showcase.
