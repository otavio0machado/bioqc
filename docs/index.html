Built for Regulated Labs

Every feature was driven by real ISO 15189 audit requirements, not invented in a spec doc.

Westgard Engine

Stateless evaluator for 1-2s, 1-3s, 2-2s, R-4s, 4-1s, 10x. Levey-Jennings charts rendered server-side via JFreeChart.

Signed PDF Reports

Hash-chained signature log — each PDF references the previous signature. QR code on the printout points to /verify/{hash} for tamper detection.

Reagent Traceability

Lot lifecycle state machine (waiting → open → in-use → end-of-use → archived). Per-unit tracking. Lot ↔ patient result chain answers any audit question in seconds.

RBAC + Audit

Role-based permissions per resource and action. Every mutation persisted with actor, timestamp, before/after diff. JWT + refresh rotation on HttpOnly cookies.

AI Commentary (Gemini)

Optional plain-Portuguese summary of monthly QC trends, appended to the regulatory report alongside deterministic statistics. Treated as advisory, never as source of truth.

Production Observability

Prometheus metrics, structured JSON logging via logstash-logback, correlation IDs propagated end-to-end. Flyway migrations with dry-run rehearsal scripts.

Pragmatic Stack

Boring, mature, well-understood. Optimized for correctness and operability, not novelty.

Case Study

From Python desktop app to web platform

Migrating a clinical lab's QC operation without disrupting daily work.

⚠ The Situation

A Brazilian clinical lab was running daily QC on a homegrown Python desktop app. Single-machine bottleneck (one analyst at a time). Reports were Word docs anyone could edit retroactively. Reagent traceability was an Excel spreadsheet. ISO 15189 surveillance was approaching.

⚙ The Action

Sole engineer on the rebuild. Spring Boot + React. Westgard rules as pure functions. Hash-chained signatures for tamper-evident reports. Reagent state machine. Paridade-first migration: ran old + new in parallel for 1-2 weeks per module before cutover.

✓ The Result

Cutover with zero analyst downtime. 15 production migrations shipped. 406 tests passing. Monthly regulatory reports auditors accept without follow-up. Multiple analysts working in parallel. Lot ↔ patient result traceability in seconds.
